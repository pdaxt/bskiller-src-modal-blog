# How Modal powered 250,000 Lovable app creations in a weekend

Source: [Modal Blog](https://modal.com/blog/lovable-case-study)

---

During a single weekend event, Lovable users built 250,000 new applications, all running in isolated development environments. Lovable used Modal to generate 1 million code sandboxes - with 20,000 running concurrently at peak - over just 48 hours.
